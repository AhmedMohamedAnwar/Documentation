### VMware , V sphere and ESXI 
**VMware:** It's the company which provide ESXI and Vsphere.
- **ESXI:** Like the engine "Hypervisor type 1" it provide the CLI only on the server.
- **Vsphere:** Hall vehicle it manage the ESXI and provide you the GUI to manage ESXI server and VMs + it's used for enterprise.
- **Mware Workstation:** It's type 2 hypervisor used for personal use for learning purpose not for enterprise.
