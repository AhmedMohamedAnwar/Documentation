### Copy & delete directory not empty 

```bash
# To copy directory contain files or other directories use -r option
# -r = recursive
cp -r dir1/ dir2/
# Delete directory not empty file by file 
rm -r dir1
# Delete directory not empty totaly by force
rm -rf dir1

```
